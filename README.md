Number Formatter
===========

A small library that adds commas to numbers

## Installation

    `npm install @mattierocks/number-formatter`

## Usage

    var numFormatter = require('@mattierocks/number-formatter');

    var formattedNum = numFormatter(35743);

Output should be `35,743`


## Tests

    `npm test`
